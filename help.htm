[1]. Q-map 서비스는?

Q-map 커뮤니티 맵핑이란?!
커뮤니티 맵핑이란 각 지역의 주민이나 커뮤니티를 구성하는 사람들이 지도라는 매개체를 통하여 관심지역에 관한 정보를 수집하는 과정을 통해 소통, 계획, 의사결정에 참여하는 일련의 활동을 말합니다.
저희 Q-map은 장애인의 접근성 개선을 위해 관련된 데이터를 수집하고 문제점을 파악하여 개선해 나가기 위한 지도 기반 커뮤니티 서비스입니다.
장애인 접근성 시설이 부족하거나 휠체어가 이동하기 어려운 방지턱등 장애인들이 불편해하는 정보들을 수집하고 이를 공공데이터로 활용함으로써 장애인 문제에 대한 인식 전환과 개선을 위해 2013년 출발했습니다.

[2]. 지도 이용 방법

  • 메뉴 클릭시 해당 메뉴 목록 왼쪽에 표시 (장소등록/지도보기/목록보기/도움말/사용자등록/로그인)
  • 목록 클릭시 텍스트 버젼의 지도 화면을 볼수 있습니다. 시각장애인은 이 화면을 바탕으로 핸드폰에 내장되어 있는 음성안내(보이스 오버, 토크백)등의 기능을 이용할 수 있습니다.
  • 지도상에 현재 나의 위치를 표시합니다. 새로운 위치 정보를 등록하기 원하시면 이 아이콘을 클릭하거나 등록하고자 하시는 위치로 드래그 합니다.
  • 위치정보를 등록한 곳에 접근성 정보를 포함한 아이콘이 나타납니다. 해당 아이콘을 클릭할 경우 상세정보를 볼 수 있는 링크의 장소명이 표시된다.
    *장애인 접근성 등급: 5가지 장애인 접근성 시설(승강기, 촉지도, 주차장, 경사로, 화장실)이 갖춰진 정보에 의해 3가지 아이콘으로 표시됩니다.
    우수:장애인 시설물 4~5개
    보통:장애인 시설물 2~3개
    미흡:장애인 시설물 0~1개
  • 지도 검색창을 통해 해당 위치명을 검색할 수 있습니다. 장소명이나 태그명을 입력하여 검색합니다. 해당 검색어의 장소명이나 태그명 검색 결과를 볼 수 있습니다.
    • 태그명 검색 : #검색어
    • 사용자 검색 : @검색어
    • 기본 검색 : 위치명 + 태그명
  • 시각장애인용 음성안내 어플을 이용하여 사용할 수 있도록 텍스트 목록 형태의 지도를 나타냅니다.
  • 안드로이드 음성안내 : 접근성 메뉴 > TalkBack 켜기, 웹 접근성 향상(웹 스크립트 설치) 허용
  • 아이폰 음성안내: 손쉬운 사용 > VoiceOver 켜기
    *기타 자세한 사용법은 해당 제조사 메뉴얼을 참고하시기 바랍니다.
  • 지도보기 : 맵핑된 지도 검색 및 위치를 찾는 지도 표시
  • 목록보기: 텍스트 버젼의 지도를 표시
  • 위치등록 : 맵핑하고자 하는 위치 정보를 등록하는 메뉴 [로그인 필요]
  • 내 등록위치 : 내가 등록한 위차 확인 [로그인 필요]
  • 도움말: 현재 화면과 같은 도움말 표시
  • 사용자 등록: 사용자 등록 화면
  • 로그인: 사용자 로그인 화면
  • 닫기: 왼쪽 메뉴 닫기
  • 기본정보 : 해당 장소의 주소와 현재 위치에서의 거리,방위각 정보를 나타내며 장소 대한 설명글을 불 수 있습니다.
    - 접근성 시설 설치현황 : 장애인 접근성 시설 5가지 항목( 승강기, 촉지도, 주차장, 경사로, 화장실)의 정보를 확인 할 수 있습니다.
    설치된 항목을 체크하면 선택한 항목은 파란색으로 표시됩니다.
    - 정정요청 : 잘못된 내용이 있으면 해당 내용을 입력하여 요청하면 등록한 사용자가 확인할 수 있습니다. [로그인 필요]
    * 정정요청한 내용은 최초 위치정보 등록자가 내용을 검토하고 오류내용을 삭제할 수 있습니다.
  • 접근성정보 : 해당 위치나, 건물에 대해 접근성 관련된 정보를 추가할 수 있습니다. [위치 등록자]
  • 사진정보: 해당 위치나, 건물에 대해 사진 정보를 추가할 수 있습니다. [위치 등록자]
  • 리뷰정보: 해당 위치나, 건물에 대해 리뷰 정보를 추가할 수 있습니다. [로그인 필요]
  • 지도버튼 클릭시 초기화면의 지도를 볼 수 있습니다.
  • 추가버튼을 클릭하여 올리고자 하는 사진을 선택합니다. 사진이 등록된 후 해당 사진을 선택하면 사진 설명글을 입력할 수 있습니다. 모든 사진 정보를 입력 한 후 올리기 버튼을 눌러 사진 정보를 반영합니다.

[3]. 위치정보 등록

  • 주소 : 현재 위치의 주소가 자동으로 입력된다. 주소가 다르다면 주소를 수정하여 위치를 조정할 수 있습니다.
  • 장소명 : 등록하고자 하는 위치의 장소명을 입력합니다. 예) 강남구 면허시험장 민원실
  • 태그 : 등록하고자 하는 위치의 태그(분류 기준)명을 입력한다. 사용자들이 찾기 쉽도록 태그명 검색을 통해 이용할 수 있습니다.
  • 위치설명 : 등록하고자 하는 장소에 대한 설명글을 입력합니다.

[4]. 사용자 등록 및 로그인

  • Q-map계정은 이메일 주소를 아이디로 활용하여 사용하도록 하였으며 사용자 등록 후 해당 위치 정보를 등록 할 수 있습니다.
    *영문 대소문자와 숫자, 특수문자 _(언더바) -(대시) .(도트) 조합 6~12자
  • 등록한 사용자 이메일 주소와 비밀번호를 입력
